Weet meer over mij

Mijn naam is Keano Koopman en Ik ben 17 jaar. Zelf heb ik een bijbaan bij een supermarkt in mijn wijk. Ik wil voor mijzelf een 3d printer halen en daarmee wil ik leuke en handige dingen verkopen om uiteindelijk bij de supermarkt weg te gaan. Ik heb deze website gemaak om mensen te informeren over sporten en normale informatie die ikzelf allemaal weet en allemaal kan.

Wat doe ik voor opleiding en wat kan ik hierdoor?

Ik doe de opleiding Software Developer en tijdens mijn opleiding heb ik 4 verschillende talen geleerd. Deze talen heten: Html, Css, Javascript en php. Html is voor alle dingen waar je op klikt, zoals: navigatie of links. Css is voor hoe de website eruit ziet met kleur en foto's. Javascript is voor het maken van de website en die interactief en dynamisch maken. Php is een open source scripttaal die aan de serverzijde wordt uitgevoerd. Het wordt gebruikt om dynamische webpagina's en applicaties te creëren die zich aanpassen aan gebruikersinvoer of databasegegevens.
